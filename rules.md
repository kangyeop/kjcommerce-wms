# Package Manager
- Use `pnpm` for all package management operations (install, add, remove, etc.).

# File Naming
- All file names should be in camelCase (e.g., `myComponent.tsx`, `utils.ts`).

# React Guidelines
- Use `sonner` for toast notifications instead of `alert`.
